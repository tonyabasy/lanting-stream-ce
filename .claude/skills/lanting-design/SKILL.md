# Lanting Stream 前端设计 Skill

## 这是什么

这套 skill 定义了 Lanting Stream 前端的唯一设计标准。
所有前端代码必须符合这套规范，不允许例外。

**风格基准：claude.ai**
- 背景：米白暖调（不是纯白）
- 品牌色：clay 橙 `#D97757`
- 标题字体：Georgia Serif
- 正文字体：system-ui Sans
- 原则：极简无装饰，留白即设计

---

## 做任何前端任务前，必读流程

```
第一步：读 tokens.md         ← 颜色、间距、圆角、字体的具体值
第二步：读 components.md     ← 组件写法和正反例
第三步：写代码
第四步：执行 checklist.md    ← 提交前必须全部通过
```

不允许跳过任何一步。

---

## 什么时候读哪个文件

| 任务类型 | 必读文件 |
|---------|---------|
| 任何前端任务 | tokens.md |
| 新建组件、修改组件 | tokens.md + components.md |
| 调整颜色、间距、字体 | tokens.md |
| 提交代码前 | checklist.md |
| 不确定某个写法对不对 | checklist.md |

---

## 设计哲学（三句话）

1. **结构清晰**：用户永远知道自己在哪里、能做什么
2. **视觉克制**：颜色只传达信息，不用于装饰；clay 橙只出现在关键位置
3. **AI 融入流程**：AI 操作在用户当前上下文里发生，不是独立窗口

---

## 最容易犯的错误（必须记住）

```
❌ 背景用纯白 #ffffff        → ✅ 用米白 #FAF9F6
❌ 文字用 #333333            → ✅ 用暖黑 #1A1915
❌ 按钮用蓝色 #1890ff        → ✅ 用 clay 橙 #D97757
❌ 顶栏用品牌色背景           → ✅ 顶栏用米白 #FAF9F6
❌ 卡片圆角用 8px            → ✅ 卡片圆角用 10px
❌ 标签用胶囊形 999px         → ✅ 标签用方形 4px
❌ 字重用 700/bold           → ✅ 字重只用 400/500
❌ 标题不加 Serif            → ✅ 标题必须加 fontFamily: 'Georgia, serif'
❌ 卡片加 box-shadow         → ✅ 卡片只用 border: 0.5px solid #E8E5DF
```
