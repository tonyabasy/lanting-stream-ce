# Editor 页面设计文档

## 概述

Editor 是 Lanting Stream 的核心页面，用于 Flink SQL 作业的开发。页面为全屏沉浸式设计，无独立顶部导航栏，最大化编辑区域。

---

## 整体布局

```
┌──────────────────────────────────────────────────────────────┐
│ [Logo]           [启停][验证]|[格式化]｜[跳转运维][跳转发布][发布] │
├──────┬───────────────┬─────────────────────────────────┬─────┤
│      │               │                                 │     │
│  ↑   │  [文件][DDL]  │                                 │  ↑  │
│      │               │                                 │     │
│ 文件 │               │                                 │ 集群│
│ DDL  │  文件目录      │         编辑器主体               │ 版本│
│      │  或            │                                 │ 配置│
│      │  DDL 列表      │                                 │ 历史│
│  ↓   │               ├─────────────────────────────────│     │
│      │               │                                 │  ↓  │
│ 运行 │               │         底部面板                 │     │
│ 版本 │               │    （运行记录 / 版本控制）          │     │
├──────┴───────────────┴─────────────────────────────────┴─────┤
│  工作空间 / 任务开发 / job_name          行24 列8  缩进:2  🔒  │
└──────────────────────────────────────────────────────────────┘
```

---

## 各区域详细说明

### 1. 顶部操作栏

整页唯一的横向顶栏，Logo 和所有操作在同一行。

**左侧（左对齐）**

| 元素 | 类型 | 说明 |
|------|------|------|
| Logo | 品牌标识 | 点击无跳转，仅作品牌展示 |
| 启停 | 状态切换按钮 | 运行中显示「停止」，已停止显示「启动」 |
| 验证 | 按钮 | 验证 SQL 语法，不执行 |
| 分隔线 | 视觉分隔 | 区分执行类和编辑类操作 |
| 格式化 | 按钮 | 格式化当前 SQL |

**右侧（右对齐）**

| 元素 | 类型 | 说明 |
|------|------|------|
| 跳转运维 | 图标按钮 | 新页面打开运维中心 |
| 跳转发布 | 图标按钮 | 新页面打开发布平台 |
| 发布 | 主操作按钮 | 发布当前作业 |

---

### 2. 左侧按钮栏

窄条按钮栏，参考 IDEA 左侧工具栏模式，按钮上下聚集分布。

```
┌──────┐
│  ↑   │
│ 文件 │  ← 点击切换第二栏到文件目录 Tab
│ DDL  │  ← 点击切换第二栏到 DDL Tab
│      │
│      │
│  ↓   │
│ 运行 │  ← 点击展开底部面板（运行记录）
│ 版本 │  ← 点击展开底部面板（版本控制）
│      │
│  🔘  │  ← 全局导航悬浮按钮（左下角）
└──────┘
```

**全局导航悬浮按钮**
- 位置：左侧按钮栏最底部
- 平时透明，几乎感知不到
- 触发方式：点击（不是 hover，避免干扰编码）
- 点击后：弹出全局导航浮层，覆盖在编辑器上方，不推开布局

---

### 3. 第二栏：文件 / DDL 区

```
┌───────────────┐
│ [文件] [DDL]  │  ← Tab 切换
├───────────────┤
│               │
│  文件目录      │  或  DDL 列表
│  （树形结构）  │
│               │
│               │
└───────────────┘
```

- 两个 Tab：「文件」显示作业文件树，「DDL」显示数据定义列表
- 宽度可拖拽调节
- 左侧按钮栏的「文件」「DDL」按钮控制此处 Tab 切换

---

### 4. 编辑器主体

占满中间剩余空间，垂直方向被底部面板压缩（面板展开时）。

编辑器本身的能力（语法高亮、自动补全等）为独立模块，不在此文档描述范围内。

---

### 5. 右侧按钮栏

参考 IDEA 右侧工具栏模式，上半部分向上聚集。

```
┌─────┐
│  ↑  │
│ 配置│  ← 作业配置（含集群、版本、运行参数），点击弹出右侧抽屉
│     │
│  ↓  │
│     │  ← 暂留空
└─────┘
```

每个按钮点击后的展开方式（浮层 / 抽屉 / 面板）待后续各功能模块设计时确定。

---

### 6. 底部面板

由左侧按钮栏的「运行记录」或「版本控制」按钮触发展开。

```
┌─────────────────────────────────────────┐
│ [运行记录] [版本控制]                    │  ← Tab 切换
├─────────────────────────────────────────┤
│                                         │
│  面板内容                                │
│                                         │
└─────────────────────────────────────────┘
```

- 高度可拖拽调节
- 展开时压缩编辑器主体高度
- 关闭时编辑器恢复全高

---

### 7. 底部状态栏

固定在页面最底部，始终可见。

```
┌──────────────────────────────────────────────────────────────┐
│  工作空间 / 任务开发 / job_name          行24 列8  缩进:2  🔒  │
└──────────────────────────────────────────────────────────────┘
```

| 元素 | 对齐 | 说明 |
|------|------|------|
| 面包屑 | 左对齐 | 当前文件的路径层级 |
| 行列号 | 右对齐 | 光标当前位置 |
| 缩进 | 右对齐 | 点击可调节缩进空格数 |
| 只读锁定 | 右对齐 | 图标状态，点击切换只读/编辑模式 |
