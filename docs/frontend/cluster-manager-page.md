# 集群管理页设计文档

## 概述

集群配置的管理页面，以卡片形式展示所有集群，支持新建、编辑、删除、连通性测试和启用/禁用。集群是独立的一级导航菜单。

---

## 整体布局

```
┌────────────────┬────────────────────────────────────────────┐
│ ⚡ Lanting     │  集群管理                    [+ 新建集群]  │

│                │                                            │
│ [首页]         │  ┌──────────────────┐ ┌──────────────────┐ │
│                │  │ prod-yarn         │ │ dev-local         │ │

│ [发布]      ›  │  │ ● 连接成功        │ │ ● 未知            │ │
│                │  │                  │ │                   │ │
│ [运维]      ›  │  │[测试][编辑][删除] │ │[测试][编辑][删除] │ │
│                │  │               ●─ │ │               ○─ │ │
│ [集群]  ←当前  │  └──────────────────┘ └──────────────────┘ │
```

`●─` 表示 Toggle 启用，`○─` 表示 Toggle 禁用（卡片整体置灰）。

## 卡片设计

### 展示信息

| 字段 | 说明 |
|------|------|
| 集群名称 | 大字，卡片标题 |
| 资源类型 · 部署模式 | 次要文字，一行展示，如「YARN · Session」 |
| Flink 版本 | 次要文字，如「Flink 2.2.0」，由 FLINK_HOME 检测后自动填入 |
| 连接状态 | 带颜色圆点 + 文字 |

### 连接状态

| 状态 | 颜色 | 说明 |
|------|------|------|
| 连接成功 | 绿色 | 上次连通性测试通过 |
| 连接失败 | 红色 | 上次连通性测试失败 |
| 未知 | 灰色 | 从未测试过 |

### 卡片操作

| 操作 | 说明 |
|------|------|
| 测试连通性 | 实时检测并刷新连接状态，期间按钮显示 loading |
| 编辑 | 打开 Modal，表单回填当前配置 |
| 删除 | 二次确认后删除 |
| 启用/禁用 Toggle | 禁用时卡片整体置灰，连接状态隐藏，不出现在作业配置的集群选择列表中 |

### 禁用状态说明

卡片禁用时：
- 整体置灰
- 连接状态隐藏
- 测试、编辑、删除操作隐藏
- 仅保留启用 Toggle

## 新建 / 编辑 Modal

### 表单联动逻辑

```
填写集群名称
    ↓
填写 FLINK_HOME → 点击「检测」按钮
    → 成功：自动填入 Flink 版本号
    → 失败：提示 FLINK_HOME 无效
    ↓
选择资源类型
    YARN       → 部署模式可选：Session / Application
    Kubernetes → 部署模式可选：Session / Application
    Local      → 部署模式固定：Mini-cluster（不需要选择）
    ↓
选择部署模式 → 动态渲染对应额外参数表单
```

### 各资源类型额外参数

**YARN Session**
- YARN Application ID
- JobManager 内存
- TaskManager 内存
- Slots 数量

**YARN Application**
- 队列名称
- JobManager 内存
- TaskManager 内存
- Slots 数量

**Kubernetes Session**
- Namespace
- Kubeconfig 路径
- JobManager URL

**Kubernetes Application**
- Namespace
- Kubeconfig 路径
- 镜像地址
- Cluster ID 前缀

**Local Mini-cluster**
- 并行度

### Modal 操作

| 操作 | 说明 |
|------|------|
| 取消 | 关闭 Modal，不保存 |
| 确认 | 保存配置 |

## 交互说明

- 删除二次确认文案：「确认删除集群「xxx」？删除后不可恢复。」
- 社区版集群数量上限 2 个，达到上限时「新建集群」按钮禁用，tooltip 提示「社区版最多支持 2 个集群」
- 编辑时表单回填所有当前配置，包括动态额外参数部分
