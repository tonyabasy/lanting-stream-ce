# 登录页设计文档

## 概述

独立页面，无顶栏无侧边栏，页面居中单卡片布局。

---

## 布局结构

```
┌─────────────────────────────────────────┐
│                                         │
│                                         │
│         ┌─────────────────────┐         │
│         │      Lanting        │         │
│         │   实时数据开发平台    │         │
│         │                     │         │
│         │  [👤 用户名        ] │         │
│         │                     │         │
│         │  [🔒 密码      👁  ] │         │
│         │                     │         │
│         │  □ 记住我            │         │
│         │                     │         │
│         │  [      登 录      ] │         │
│         └─────────────────────┘         │
│                                         │
│                                         │
└─────────────────────────────────────────┘
```

---

## 各元素说明

| 元素 | 说明 |
|------|------|
| 页面背景 | 浅灰色 |
| 卡片 | 白色，居中，圆角 |
| 品牌名「Lanting」 | 卡片顶部，加粗大字 |
| 副标题「实时数据开发平台」 | 品牌名下方，次要文字色 |
| 用户名输入框 | 左侧用户图标，placeholder「用户名」 |
| 密码输入框 | 左侧锁图标，右侧显示/隐藏切换图标 |
| 记住我 | Checkbox，默认未选中 |
| 登录按钮 | 全宽，主色背景，白色文字 |

---

## 交互说明

- 支持回车提交
- 登录失败通过 message 全局提示错误信息，不在表单内展示
- 登录成功跳转首页，若有 `redirect` 参数则跳转至对应页面
